# zensvg

SVG rendering and lossless optimization with zencodec integration.

## Features

- **Render** SVG/SVGZ to RGBA8 pixels via [resvg](https://github.com/nickel-graphics/resvg)
- **Optimize** SVGs losslessly: strip comments, metadata, whitespace; compress to SVGZ
- **zencodec** decoder trait integration for use in zen image pipelines
- Configurable DPI, scale, target dimensions, fit modes
- Resource limits (max pixels, max dimensions)

## Usage

```rust
use zensvg::{render, RenderOptions};

let svg_data = br#"<svg xmlns="http://www.w3.org/2000/svg" width="100" height="100">
    <circle cx="50" cy="50" r="40" fill="red"/>
</svg>"#;

let output = render(svg_data, &RenderOptions::default()).unwrap();
assert_eq!(output.width, 100);
assert_eq!(output.height, 100);
// output.data contains RGBA8 pixels with straight alpha
```

### SVG Optimization

```rust
use zensvg::{optimize, OptimizeOptions};

let svg = br#"<?xml version="1.0"?>
<!-- Generated by editor -->
<svg xmlns="http://www.w3.org/2000/svg">
    <metadata><rdf>...</rdf></metadata>
    <rect width="100" height="100" fill="blue"/>
</svg>"#;

// Minify: strip comments, metadata, whitespace
let minified = optimize(svg, &OptimizeOptions::minify()).unwrap();

// Maximum compression: minify + SVGZ
let compressed = optimize(svg, &OptimizeOptions::max_compression()).unwrap();
```

## Feature Flags

| Feature | Default | Description |
|---------|---------|-------------|
| `std` | yes | Standard library support |
| `text` | yes | Text rendering (loads system fonts) |
| `zencodec` | no | zencodec decoder trait integration |
| `optimize` | no | SVG lossless optimization and SVGZ compression |
